# Scalable Data Processing Pipeline

This repository demonstrates scalable processing of large structured datasets
using Dask for distributed computation and Pandas for analytical summarisation.

## Architecture

- Modular design
- Distributed Parquet loading
- Memory-efficient transformations
- Summary analytics
- CI pipeline with testing

## Technologies

- Python 3.10
- Dask
- Pandas
- PyArrow
- GitHub Actions CI

## Scalability Approach

- Partition-aware loading
- Lazy computation using Dask
- Separation of concerns (loader, preprocessing, analytics)
- CI-enforced code quality

## Future Improvements

- BigQuery ingestion module
- Docker containerization
- Logging & metrics integration
